---
title: Start tracking
---

To track a channel, you can use the `/track` command. You can select a channel by either typing it's name and selecting it among the options.

![Autocomplete](/docs/channel-autocomplete-track.png)

Or by copy-pasting their channel ID.

## Ping role

If you want a role to be pinged after an update is sent, specify a role in the `ping_role` option.

## Any problems?

If you don't see a message like that, make sure to check `/tracking` to see if it is on the list. If it doesn't, make sure to contact us on [Discord](/discord).
